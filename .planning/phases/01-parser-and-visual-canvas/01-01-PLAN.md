    Create `tools/er-editor/parser/extractor.py` with:

    1. Import libcst, the IR dataclasses from `.ir`, and `dataclasses.asdict`

    2. Create `class ModelExtractor(cst.CSTVisitor)`:
       - `__init__`: self.tables = [], self._current_class = None, self._in_target_class = False, self._found_tablename = False

       - `visit_ClassDef`: Set self._current_class to a new TableIR(class_name=node.name.value, table_name=""). Set self._in_target_class = True, self._found_tablename = False. Return None (visit children).

       - `leave_ClassDef`: If self._current_class and self._found_tablename (meaning __tablename__ was found), append to self.tables. Reset state. This is the key detection: only classes WITH __tablename__ become tables.

       - `visit_SimpleStatementLine` / `visit_AnnAssign` / `visit_Assign`: Inside a target class, detect:
         a) `__tablename__` assignment -> extract string value, set self._current_class.table_name and self._found_tablename = True
         b) `__table_args__` assignment -> parse for {"schema": "..."} pattern. Walk the Dict node looking for a key "schema".
         c) `Mapped[...] = mapped_column(...)` -> create ColumnIR:
            - name: from the target (left side of assignment)
            - type: first positional arg of mapped_column() call. If it's a Name node, use node.value (e.g. "Integer"). If it's a Call node, reconstruct as "TypeName(args)" (e.g. "String(50)", "DateTime(timezone=True)"). If no positional arg, fall back to extracting from Mapped[] annotation.
            - primary_key: look for keyword arg primary_key=True in mapped_column()
            - nullable: check Mapped annotation for union with None (e.g. `str | None`). If found, default True. Then check mapped_column() for explicit nullable= keyword. Explicit overrides annotation.
            - foreign_key: look inside mapped_column() args for a ForeignKey() call. Extract the string argument.
            - unique: keyword arg unique=True in mapped_column()
            - index: keyword arg index=True in mapped_column()
            - default: keyword arg default=X in mapped_column() -> str(X)
            - server_default: keyword arg server_default=X in mapped_column() -> extract the inner string from text("...") or str(X)
         d) `relationship(...)` assignment -> create RelationshipIR:
            - name: from the target (left side)
            - target: first positional string arg of relationship(), OR extract from Mapped[list["Target"]] or Mapped["Target"] annotation
            - back_populates: keyword arg
            - cascade: keyword arg
            - lazy: keyword arg
            - uselist: check Mapped annotation for list[X] -> True, bare X -> False. Explicit uselist= keyword overrides.
            - order_by: keyword arg

    3. Create helper methods on the visitor:
       - `_get_keyword_arg(call_node, keyword_name) -> cst.BaseExpression | None`: find a keyword arg by name in a Call node
       - `_get_keyword_bool(call_node, keyword_name) -> bool | None`: extract boolean from keyword arg
       - `_get_keyword_str(call_node, keyword_name) -> str | None`: extract string from keyword arg
       - `_extract_mapped_type(annotation) -> tuple[str, bool]`: from Mapped[X] or Mapped[X | None], return (type_string, is_nullable)
       - `_reconstruct_type(node) -> str`: turn a CST expression into a display string like "String(50)" or "Integer"
       - `_is_relationship_call(node) -> bool`: check if the value is a relationship() call
       - `_is_mapped_column_call(node) -> bool`: check if the value is a mapped_column() call

    4. Create top-level function:
       ```python
       def extract_model(source: str) -> list[TableIR]:
           tree = cst.parse_module(source)
           extractor = ModelExtractor()
           tree.walk(extractor)
           return extractor.tables
       ```
